Whitecape Backup Platform Overview

Secure, multi-tenant backup and restore platform powered by Restic and Ansible

Advanced Security
End-to-end encryption with AES-256, secure key management, and isolated tenant data
Flexible Deployment
No agents required on target systems - just SSH access for efficient backup orchestration
Efficient Storage
Deduplication and incremental backups optimize bandwidth usage and reduce storage costs

Key Platform Features

  • Multiple Backup Types: Full, differential, and incremental (Restic) options for maximum flexibility
  • Role-Based Access: Owner/Admin and Operator roles with appropriate permissions for each
  • Automated Scheduling: Set up recurring backup jobs with customizable retention policies
  • Real-time Monitoring: Track job status, view logs, and receive notifications on backup health
  • Azure Storage Integration: Direct support for Azure Blob Storage with optimized transfers
  • Rapid Restore: Easily browse and restore specific files or entire backups from the web interface
Multi-Tenant Architecture
Complete isolation between tenants with customizable policies and quotas for each
Analytics Dashboard
Visualize backup success rates, storage usage, and system health metrics
Ansible Integration
Leverage the power of Ansible playbooks for customizable backup operations
