Getting Started Guide

Set up your backup environment in just a few simple steps

1

Create Account

Sign up for a new account to become the administrator of your organization's tenant. You'll have full access to all features and be able to add additional users as needed.

Your account provides access to the admin dashboard with complete analytics and management capabilities.
2

Configure Target Servers

Add your servers by providing their connection details. The system requires:

  • Server hostname or IP address
  • SSH username for connection
  • SSH public key for authentication
Ensure your SSH keys are properly secured. The system uses these keys to perform agentless backups.
3

Set Up Storage

Configure your backup storage destination. You have two primary options:

Azure Blob Storage

Cloud-based storage with scalable capacity and geo-redundancy options

Local Storage

On-premises storage for organizations with specific compliance needs

4

Create Backup Job

Create your first backup job through the dashboard:

  1. Select the target server
  2. Choose backup type (Full, Differential, or Incremental/Restic)
  3. Define the backup schedule (if automated)
  4. Select storage destination
  5. Launch the job
Your first backup will establish the baseline for future incremental or differential backups.
5

Manage Users

Add additional users to your tenant as needed:

  • Admin users have full access to all features and settings
  • Operator users can perform backup and restore operations but have limited administrative access
Use role-based access to implement least privilege security principles in your organization.
